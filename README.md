## 文件结构
```shell
.
├── public  打包所需静态资源
└── src
    ├── untils  axios封装
    └── assets  项目静态资源
        ├── iconfont  自定义图标资源[http://www.iconfont.cn]
        ├── img  图片资源
        ├── lang  i18n用语言包
        └── css  全站样式
    ├── components  公共组件
    ├── home  欢迎页
    ├── about  个人简介
    ├── skill  十八般武艺
    ├── holyland  用武圣地
    ├── formula  口诀心法
    └── tools  如意兵器谱
```
